# How to run the project

## Requirements

Node: v12.16.1
Yarn: v1.16.0
Typescript: v3.8.0

<!-- or Docker -->

### Getting started

Clone the repo

- git clone x

### Backend

```bash
yarn && cp .env.example .env && yarn start
```

### Frontend

```bash
yarn && cp ./public/env.example . ./public/env && yarn start
```

To install dependencies

`yarn`

To set up any env config

`cp .env.example .env`

### Running

Running for development

`yarn watch`

Running locally

`yarn dev`

Running production

tbc

### Linter

yarn lint

yarn test
